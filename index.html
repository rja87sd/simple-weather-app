Week 6 Day 5 Assignment | Building an App

Instructions

Part 1: Setup Your Project

  1. Name your repository simple-weather-app.
  2. Initialize it with index.html, script.js, and style.css files.

Part 2: Define the Weather Class

  1. In script.js, define a Weather class with a constructor that sets up properties such as temperature, description, and any other relevant data.
  2. Include methods for processing and displaying weather data.

Part 3: Interface for Weather Data Input

  1. In index.html, create a simple form that allows users to input a location (city or zipcode).
  2. Add a button to submit the form and fetch weather data.

Part 4: Fetch and Display Weather Data

  1. Use a weather API like Weatherbit to fetch weather data based on user input.
  2. Process the API response by creating an instance of your Weather class and display the data in a user-friendly format in your HTML.

Part 5: Styling and Testing

  1. Use style.css to style your weather app. Focus on creating a clean and readable layout.

Part 6: Deployment and Submission

  1. Push your code to GitHub and deploy your project using GitHub Pages or a similar service.
  2. Provide the URL link to your GitHub repository and the live deployment as your assignment submission.
